# CV Website Profesional (Struktur Modern)

Template CV berbasis web dengan struktur folder profesional seperti project frontend modern.

## Struktur Folder
```
project/
│── public/        # file publik (favicon, thumbnail)
│── src/           # source code utama
│   │── css/
│   │── js/
│   │── assets/
│
│── index.html
│── README.md
```

## Fitur
- Struktur clean (mirip project industri)
- SEO + Open Graph (thumbnail share)
- Favicon di tab browser
- Siap deploy

## Cara Pakai
1. Edit file index.html
2. Ubah isi CV sesuai data
3. Ganti gambar di folder public/
4. Deploy ke hosting

## Rekomendasi Hosting
- GitHub Pages
- Netlify
- Vercel

## Catatan
Struktur ini scalable, jadi bisa dikembangkan ke React / Next.js nanti.
